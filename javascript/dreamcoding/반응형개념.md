# 반응형개념

> [mdn media queries](https://developer.mozilla.org/ko/docs/Web/Guide/CSS/Media_queries)
>
> `mobile` : 320~480px
>
> `tablet` : 768~1024px
>
> `desktop` : 1024px이상
>
> css의 `@media`쿼리가 반응형을 도와줌
>
> ```css
> /*
> screen 대신 speech/print/all 쓸수있음
> and 대신 not/only/,(comma) 쓸수있음
> */
> @media screen and (min-width:800px) {
>     .container {
>         width:50%;
>     }
> }
> ```
