## ì´ì§„íƒìƒ‰(SWEA_4839)

- ì´ì§„íƒìƒ‰ì„ í•¨ìˆ˜ë¡œ ë§Œë“¤ì—ˆë‹¤, ì²˜ìŒì—ëŠ” keyê°’ë³´ë‹¤ í¬ê±°ë‚˜ ì‘ì„ë•Œ endê°’ê³¼ startê°’ì„ middle-1,middle+1ë¡œ ì¤˜ì„œ ì˜¤ë¥˜ê°€ ë‚¬ë‹¤. ì•„ë§ˆ ê°’ì„ 1í¬ê²Œ í•˜ë”ë¼ë„ ë‚˜ì¤‘ì— middleì„ êµ¬í•˜ëŠ” ê³¼ì • ì¤‘ ì˜¤ë¥˜ê°€ ë‚˜ê¸° ë•Œë¬¸ì¸ ê²ƒê°™ë‹¤.

```python
#ì´ì§„íƒìƒ‰ì„ ì´ìš©
#ì „ì²´ ìª½ìˆ˜ì™€ Aì™€ Bê°€ ì°¾ì•„ì•¼ë  ìˆ«ìë¥¼ ì…ë ¥ë°›ëŠ”ë‹¤
#Aì˜ ì´ì§„íƒìƒ‰ê³¼ Bì˜ ì´ì§„íƒìƒ‰ íšŸìˆ˜ë¥¼ ë¹„êµí•´ì„œ ì ì€ ì‚¬ëŒì´ ì´ê¸´ë‹¤, ë¹„ê¸°ë©´ 0

def bin_search(P,key):
    start = 1
    end = P
    cnt = 0
    while start <= end:
        cnt += 1
        middle = (start + end) // 2

        #key ê°’ê³¼ ê°™ì€ ê²½ìš°
        if middle == key:
            return cnt
        #keyê°’ë³´ë‹¤ í° ê²½ìš°
        elif middle > key:
            end = middle
        #keyê°’ë³´ë‹¤ ì‘ì€ ê²½ìš°
        else:
            start = middle


T = int(input())
for tc in range(1,T+1):
    P, Pa, Pb = map(int,input().split())

    A = bin_search(P,Pa)
    B = bin_search(P,Pb)

    #Aì˜ cntê°€ Bë³´ë‹¤ í´ë•Œ
    if A > B:
        print(f'#{tc} B')
    elif A == B:
        print(f'#{tc} 0')
    else:
        print(f'#{tc} A')
```
